# Ebru · Su Üstünde Mürekkep

Tarayıcıda gerçek zamanlı akışkan simülasyonuyla ebru / su üstünde mürekkep sanatı.
Tek dosya (`index.html`), sıfır bağımlılık, saf WebGL2.

## Canlı

https://ceressa.github.io/ebru/

## Nasıl çalışır

- Navier-Stokes tabanlı GPU akışkan simülasyonu: advection, vorticity (girdap), basınç çözümü
- Mürekkep, kağıt üzerine soğurma (Beer-Lambert) modeliyle çizilir; renkler doğal biçimde akar
- Dokun: mürekkep damlat · Sürükle: suyu karıştır, deseni akıt
- Otomatik akış (✦): boştayken kendi kendine damlatıp suyu yavaşça döndürür
- 7 ebru rengi (lacivert, çini mavi, kırmızı, sarı, yeşil, mor, siyah)
- Açılıp kapanabilen ortam sesi, PNG indir, temizle
